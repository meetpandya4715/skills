# codex-skills

Standalone repository for custom Codex skills.

## Included Skills

- `repo-x-ray`: inspects a repository and generates evidence-backed Mermaid diagrams for fast codebase understanding.

## Layout

```text
skills/
  repo-x-ray/
    SKILL.md
    agents/
      openai.yaml
```

## Usage

Copy a skill folder into your Codex skills directory or use this repository as a source when publishing and sharing skills.
